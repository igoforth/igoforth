### Ian Goforth

Security researcher building ML-driven tools for vulnerability research and reverse engineering.

Previously at Battelle doing RF/EW vulnerability research and 5G security. OSCP certified.

#### Recent work

- [aletheia](https://github.com/igoforth/aletheia) - AI decompiler targeting MLIR. Lifts binary to structured control flow IR, deterministic passes lower to C. [Blog post](https://ian.goforth.systems/blog/aletheia).
- [excido](https://github.com/igoforth/excido) - Fuzzing harness generator that resolves C/C++ type dependencies across translation units via Clang AST. [Blog post](https://ian.goforth.systems/blog/excido).
- [bistatic-doppler-localization](https://github.com/igoforth/bistatic-doppler-localization) - Target localization from multi-transmitter bistatic Doppler shifts using neural networks with a custom multi-term loss for sparse position targets. [Blog post](https://ian.goforth.systems/blog/bistatic-doppler-localization).
- [broll-search](https://github.com/igoforth/broll-search) - B-roll footage search using scene detection, visual embeddings, and natural language captioning. [Blog post](https://ian.goforth.systems/blog/broll-search).
- [faire-auth](https://github.com/igoforth/faire-auth) - Auth library rebuilt on Hono with typed middleware, route hooks, and DTO transforms. Runs on Node, Bun, Deno, Cloudflare Workers, and edge runtimes. [Blog post](https://ian.goforth.systems/blog/faire-auth).
